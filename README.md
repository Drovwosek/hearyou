## Что умеет сервис

### 1. Транскрибация аудио/видео
- Русский, английский, турецкий языки
- Точность: 92-95% для русского
- Форматы: MP3, WAV, OGG, M4A, FLAC, MP4, AVI, MOV, MKV
- Размер: до 25 GB, любая длительность

### 2. Определение спикеров (НОВОЕ!)
- Автоматическое разделение "кто говорит"
- Точность: 75-85%
- Формат: диалог с метками SPEAKER_00, SPEAKER_01

### 3. Улучшение качества
- 40+ специальных терминов (iSpring, СПРИНТ, ЕГЭ)
- 32 автоисправления
- Фильтр слов-паразитов
- Литературный текст (без "эээ", "ммм")

## Технологии

- **STT:** локальный Whisper (`faster-whisper`) без внешней STT-зависимости
- **Диаризация:** Resemblyzer (voice embeddings)
- **Backend:** Python + FastAPI
- **Frontend:** HTML5 + JavaScript (SSE)
- **Infrastructure:** Docker + Nginx + Redis + PostgreSQL

## FAQ

**Q: Можно ли обработать видеозвонок Zoom?**  
A: Да! Загрузите запись, аудио извлечётся автоматически.

**Q: Сколько спикеров определяется?**  
A: По умолчанию 2. Можно настроить в коде.

**Q: Работает ли оффлайн?**  
A: Да, сама транскрибация работает локально. Интернет может понадобиться только при первом скачивании Whisper-модели.

**Q: Сохраняются ли файлы?**  
A: Нет, удаляются после обработки.
