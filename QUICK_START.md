# 🚀 HearYou - Быстрый старт

**Сервис транскрибации с улучшением качества текста**

---

## 🌐 Веб-интерфейс

**URL:** http://92.51.36.233:8000/

### Как использовать

1. **Откройте браузер** → http://92.51.36.233:8000/
2. **Загрузите аудиофайл** (MP3, WAV, AAC, OGG и др.)
3. **Настройте параметры:**
   - ✅ **Literature text** - убирает базовые звуки (эээ, ммм)
   - ✅ **Corrections** - исправляет артефакты (иишка → ИИшка, егэ → ЕГЭ)
   - ⬜ **Clean filler words** - убирает слова-паразиты (вот, типа, короче)
   - ⬜ **Определить спикеров** - кто и когда говорил
4. **Нажмите "Загрузить и транскрибировать"**
5. **Получите результат** - чистый текст транскрипции

## 💰 Стоимость

### Local Whisper (транскрипция)
- **БЕСПЛАТНО за минуту аудио** - работает локально через `faster-whisper`
- По умолчанию: `WHISPER_MODEL=small`, `WHISPER_DEVICE=cpu`, `WHISPER_COMPUTE_TYPE=int8`
- Для первого запуска может понадобиться интернет, чтобы скачать модель

### Улучшение качества (TextCleaner)
- **БЕСПЛАТНО** - работает локально
- Чистая логика на Python (regex + словари)
- Никаких API вызовов

### JTBD анализ
- **ОТКЛЮЧЕН** по умолчанию
- Не тратит токены, пока не включён
- При необходимости - обсудить варианты

## ❓ FAQ

### Почему чекбокс JTBD неактивен?
JTBD анализ отключен по умолчанию, так как требует платный API. Основные функции (транскрипция + улучшение качества) работают полностью.

### Сколько времени занимает транскрибация?
- 1 минута аудио ≈ 5-10 секунд обработки
- 30 минут аудио ≈ 2-5 минут
- Speaker diarization добавляет +30-50% времени

### Можно ли скачать результат?
Да, результат можно скопировать из веб-интерфейса. API endpoint `/transcribe` возвращает JSON.

### Работает ли офлайн?
Да, после скачивания Whisper-модели транскрибация и улучшение качества работают локально.
