
### Категория: Jobs (Работы)

```
┌─────────────────────────────────────────────────────┐
│ 🎯  Работы (Jobs)                            [3]   │ ← Синяя полоска слева
└─────────────────────────────────────────────────────┘

  • Понять возможности электронного журнала для
    управления учебным процессом
    ████░ 92%  ← зелёный прогресс-бар (высокая уверенность)

  • Оценить совместимость СПРИНТ с текущей LMS
    инфраструктурой
    ████░ 88%  ← зелёный прогресс-бар

  • Узнать опыт других вузов с внедрением системы
    ████░ 85%  ← зелёный прогресс-бар
```

### Категория: Pains (Боли)

```
┌─────────────────────────────────────────────────────┐
│ 😖  Боли (Pains)                             [2]   │ ← Красная полоска слева
└─────────────────────────────────────────────────────┘

  • Сложность интеграции с существующими системами
    управления обучением
    ███░░ 78%  ← оранжевый прогресс-бар (средняя уверенность)

  • Неопределенность в требованиях к адаптации под
    специфику вуза
    ███░░ 72%  ← оранжевый прогресс-бар
```

### Категория: Gains (Выгоды)

```
┌─────────────────────────────────────────────────────┐
│ 🎁  Выгоды (Gains)                           [4]   │ ← Зелёная полоска слева
└─────────────────────────────────────────────────────┘

  • Автоматизация ведения электронного журнала и
    учета посещаемости
    █████ 95%  ← зелёный прогресс-бар

  • Сокращение времени на формирование отчетности
    в 3 раза
    ████░ 89%  ← зелёный прогресс-бар

  • Гибкая интеграция с популярными LMS через API
    ████░ 91%  ← зелёный прогресс-бар

  • Адаптация под специфику конкретного учебного
    заведения
    ████░ 83%  ← зелёный прогресс-бар
```

### Категория: Context (Контекст)

```
┌─────────────────────────────────────────────────────┐
│ 📍  Контекст                                 [2]   │ ← Жёлтая полоска слева
└─────────────────────────────────────────────────────┘

  • Высшее учебное заведение с существующей LMS
    инфраструктурой
    █████ 94%  ← зелёный прогресс-бар

  • Процесс оценки и выбора системы управления
    образовательным процессом
    ███░░ 76%  ← оранжевый прогресс-бар
```

### Категория: Triggers (Триггеры)

```
┌─────────────────────────────────────────────────────┐
│ ⚡  Триггеры                                 [2]   │ ← Фиолетовая полоска слева
└─────────────────────────────────────────────────────┘

  • Необходимость модернизации системы управления
    учебным процессом
    ███░░ 68%  ← оранжевый прогресс-бар

  • Интерес к опыту других образовательных
    учреждений (академия Штиглица)
    ████░ 82%  ← зелёный прогресс-бар
```

## 🎨 Цветовая палитра

### Категории (Border-left цвет)

```css
🎯 Jobs     → #2563eb (синий)
😖 Pains    → #dc2626 (красный)
🎁 Gains    → #16a34a (зелёный)
📍 Context  → #ca8a04 (жёлтый)
⚡ Triggers → #9333ea (фиолетовый)
```

### Confidence градации

```css
≥80%   → Зелёный  #16a34a  ████░
50-79% → Оранжевый #d97706  ███░░
<50%   → Красный   #dc2626  ██░░░
```

### Фоновые цвета

```css
Категория header → gradient(#f9fafb, #f3f4f6)
Элемент default  → #f9fafb
Элемент hover    → #f3f4f6
Border default   → #e5e7eb
Border hover     → #d1d5db
```

## ✨ Анимации и эффекты

### 1. Появление категорий

```
fadeIn: 0.4s ease-in
  from: opacity 0, translateY(10px)
  to:   opacity 1, translateY(0)
```

### 2. Hover на элементах

```
.jtbd-item:hover {
  background: #f3f4f6
  border-color: #d1d5db
  transform: translateX(4px)  ← сдвиг вправо
}
```

### 3. Заполнение прогресс-бара

```
transition: width 0.3s ease
```

### 4. Кнопка "Скопировано"

```
Click → "✓ Скопировано!" (2 секунды)
      → возврат к "📋 Скопировать"
```
